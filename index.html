Mis proyectos

Un vistazo a algunos de mis proyectos extracurriculares que más me identifican

QR para Restaurantes

Desarrollé una plataforma que permite a restaurantes digitalizar sus menús y generar códigos QR personalizados. El sistema incluye gestión de usuarios, base de datos segura y comunicación en tiempo real entre clientes y personal. Esta solución permite a los dueños publicar sus menús y usar codigos QR únicos para cada menú.
Tecnologías: HTML, CSS, JavaScript, Firebase

Fútbol Daily: Minijuegos web diarios

Creé una colección de minijuegos diarios de fútbol, incluyendo retos tipo Wordle, crucigramas y juegos originales. Cada juego está diseñado para ser intuitivo y entretenido, con contenido que cambia automáticamente cada día.
Para la base de datos descargué datasets de Kaggle, los limpié y transformé con scripts para generar JSON que alimenta los juegos.
Tecnologías: HTML, CSS, JavaScript, Bash Scripting, JSON

Allganize: Organización inteligente para estudiantes

Allganize nace para agrupar en una sola interfaz las herramientas que un estudiante usa a diario: calendario, tableros Kanban, temporizador Pomodoro y notas para ideas y apuntes. La app prioriza usabilidad y rendimiento: funciona totalmente en el navegador, guardando datos localmente
Tecnologías: HTML, CSS, JavaScript, LocalStorage, Drag & Drop API, Web APIs
